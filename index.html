TinyPoker

Play poker on any messaging app, with hole card secrecy!

By using (terrible) cryptography, neither player can see what the other's holding is. No direct connection needed!

Assign player numbers 1 and 2, select a host and pick a 5-digit magic number, then tell that number to the host. After each phase the host gets a new code, copy-paste that into the text box and hit "Process input."

Hosting is easy, just enter both magic numbers into the controls at the bottom and cycle through the phases. The host too must enter and process the codes!

Source code available on GitHub.
